## 📱 Deskripsi Aplikasi

### Headline Utama
**Recehan - Aplikasi Kasir Pintar untuk Warung Indonesia**

Revolusi cara Anda mengelola warung dengan teknologi terdepan! Recehan hadir sebagai solusi kasir digital yang dirancang khusus untuk warung, toko kelontong, dan UMKM Indonesia.

### Value Proposition
✨ **Mengapa Recehan?**
- 🎯 **Mudah Digunakan** - Interface sederhana, cocok untuk semua usia
- 💰 **Tingkatkan Omzet** - Kelola stok dan hutang dengan rapi
- 📊 **Laporan Lengkap** - Analisis penjualan real-time
- 🔄 **Offline Ready** - Tetap bisa digunakan tanpa internet
- 💳 **Multi Payment** - Tunai, QRIS, dan sistem hutang

## 🌟 Fitur Unggulan

### 🏪 **Kasir Pintar**
- **Scanner Visual Produk** - Foto produk jadi shortcut kasir
- **Mode Ramai** - Tombol besar untuk warung sibuk
- **Saran Kembalian** - Hitung pecahan uang otomatis
- **3 Metode Bayar** - Tunai, QRIS, Hutang

### 📦 **Manajemen Produk**
- **Import CSV Massal** - Upload ratusan produk sekaligus
- **Stok Otomatis** - Alert stok menipis
- **Label Harga** - Cetak stiker harga profesional
- **Kategori Fleksibel** - Atur produk sesuai kebutuhan

### 💳 **Sistem Hutang Canggih**
- **Kasbon Cepat** - Input hutang dalam 10 detik
- **Reminder WhatsApp** - Ingatkan pelanggan otomatis
- **Riwayat Lengkap** - Track semua transaksi hutang
- **Jatuh Tempo** - Kalender Indonesia yang mudah

### 🧾 **Struk Kustomisasi**
- **5 Template Cantik** - Klasik, Modern, Lucu, Profesional, Promo
- **Logo Toko** - Branding personal di setiap struk
- **Kirim WhatsApp** - Share struk langsung ke pelanggan
- **Simpan PDF** - Backup digital otomatis

### 🌙 **Mode Adaptif**
- **Mode Malam** - Hemat mata, cocok buka 24 jam
- **Mode Senyap** - Tanpa suara untuk area sensitif
- **Hemat Baterai** - Tahan lama di HP kentang
- **Mode Ramai** - UI besar untuk warung padat
